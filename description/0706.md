- 사용자가 특정 키워드를 입력 시, 구글 SERP 내 '관련 검색어'를 호출 버튼 클릭 시 자동으로 가져온다.
- 예를 들어, '버티컬 마우스'라는 키워드의 최신 수집 SERP를 보면 다음 키워드가 관련 검색어로 노출된다.
  - 버티컬 마우스의 치명적인 단점
  - 버티컬 마우스 추천
  - 로지텍 버티컬 마우스
  - 버티컬 마우스 추천 여자
  - 버티컬 마우스 끝판 왕
  - 버티컬 마우스 디시

## 2. 브랜드/카테고리를 입력하여 Suggested Prompt 후보군 자동 생성

### 설명

- EntityLoop 페이지에서 따온 기능으로 'description/entityloop_cep.md' 파일에 관련 설명이 있습니다.

### 프로세스 제안

1. 해당 입력이 가능한 Tab이 필요합니다. 현재는 'CEP 프로젝트 선택/내 브랜드 제품정보 입력/내 콘텐츠 URL 입력'만 있는데, 브랜드와 카테고리만으로 검색할 수 있는 섹션으로 전환할 수 있는 Tab이 생기는거죠.
2. 여기서 브랜드로 '로지텍'을 입력하고 카테고리로 '마우스'라는 키워드를 입력했다고 칩시다.
3. 여기서 '카테고리에서 CEP 발굴'이라는 버튼을 클릭하면 후보 8개가 출력 됩니다.
4. 실제 엔티티 루프 기능에선 다음 6가지 CEP가 나타났어요.

- 로지텍 소프트웨어 호환성 문제: 새로운 로지텍 마우스를 구매했지만, 기존 소프트웨어와 호환되지 않아 설정에 어려움을 겪고 있다.
- 무선 마우스 배터리 수명: 무선 마우스를 사용 중인데, 배터리가 빨리 닳아 자주 교체해야 하는 불편함이 있다.
- 로지텍 g hub 사용법: 로지텍 g hub를 설치했지만, 복잡한 인터페이스 때문에 원하는 설정을 찾기 어렵다.
- 유선 마우스의 케이블 걸림 문제: 유선 마우스를 사용 중인데, 케이블이 걸려 작업에 방해가 된다.
- 로지텍 마우스 드라이버 문제: 로지텍 마우스를 사용 중인데, 드라이버 문제로 마우스가 제대로 작동하지 않는다.
- 게임용 마우스 선택의 어려움: 게임을 위한 마우스를 선택하려고 하는데, 어떤 제품이 좋은지 판단이 어렵다.
